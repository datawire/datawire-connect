Installation¶

The Datawire Connect Cloud Tools has its own installation process separate from other components of Datawire Connect.

Prerequisites¶

Datawire Connect requires a modern Linux distribution or Mac OS X. It has been successfully tested on the following:

  • Mac OS 10.10 (Yosemite)
  • Mac OS 10.11 (El Capitan)
  • Ubuntu 14.04 (Trusty)
  • Fedora 22

Datawire Connect also requires the following:

  • Python 2.7+
  • pip

Installation¶

To install Datawire Connect Cloud Tools, run the following command inside of a virtual environment:

pip install datawire-cloudtools
