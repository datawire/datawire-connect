Datawire Connect relies on the Quark language compiler. The simplest way to
install it is to go to the main Datawire Connect directory and run

make install-quark

which will install it in $HOME/.quark and offer to update your .bashrc for you.

If you'd prefer, you can install it yourself:

curl -# -L https://raw.githubusercontent.com/datawire/quark/master/install.sh | bash -s --
